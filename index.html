Viajando Ando - Sistema de Vuelos 

2020

El sistema permite al cliente buscar vuelos de líneas aéreas en fechas específicas y elegir un vuelo en función de algún detalle, como la disponibilidad. El cliente también puede reservar un vuelo o cancelarlo. El sistema muestra todos los detalles del vuelo, como el número de vuelo, el nombre, el precio y la duración del viaje. Cuando el sistema muestra la lista de vuelos disponibles, el cliente puede elegir un vuelo en particular y reservar un asiento. Para reservar un vuelo, el sistema le pide al cliente que inicie sesión si ya se ha registrado. El sistema también le permite al cliente cancelar su reserva, si surge algún problema.

El sistema es capaz de:

1. Añadir un nuevo usuario al sistema de reservación.

2. Buscar vuelos de aerolínea en una fecha específica.

3. Elegir un vuelo en base a los detalles (precio, duración, etc.)

4. Mostrar todos los detalles del vuelo elegido.

5. Contar los kilómetros acumulados de cada pasajero. Si tiene más de 50,000 km, entonces el sistema aplica descuento del 40% en el precio del vuelo.

6. Colocar en lista de espera.

7. Cancelar una reservación.

Pollution Sensors

2020

Factores como aumento de capacidades de procesamiento, mejoras en aprovechamiento de energía, etc., han dado paso a que dispositivos de procesamiento digital con sensores, permitan la transformación digital. Teniendo en cuenta lo anterior, se propone implementar un sistema con múltiples sensores y protocolos, que realicen un constante análisis de la calidad del aire en zonas más concurridas o de mayor población de la Ciudad de México, pasando estos datos a una base de datos. Con esto, el usuario podrá monitorear los datos desde una página web en tiempo real.
